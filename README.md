# Snake

A snake game I made in about an hour just for fun.

## Installation

All you need to do is use [GIT](https://git-scm.com/) to download it.

```bash
git clone https://github.com/Corplexus/basic_snake.git
```

## Usage
All you have to do is run it.

```python
python snake.py
```
